# 🚀 Imran's Arch Linux Dotfiles

Welcome to my personal dotfiles repository! This backup contains my configuration files for a heavily customized Arch Linux environment.

## 🛠️ System Details
* **OS:** Arch Linux
* **Window Manager:** Hyprland
* **Terminal:** Alacritty / Kitty / Ghostty (with Typewriter animation aesthetic)
* **Compositor/Aesthetics:** Custom frosted glass effects, rounded corners, and smooth animations.

## 📥 How to Restore (Installation)

If you are setting up a new system, simply clone this repository and copy the configurations to your `~/.config` folder:

```bash
# Clone the repository
git clone [https://github.com/imran-engz/dotfiles.git](https://github.com/imran-engz/dotfiles.git)

# Copy the configs to your local directory
cp -r dotfiles/.config/* ~/.config/
cp dotfiles/.bashrc ~/
cp dotfiles/.zshrc ~/
